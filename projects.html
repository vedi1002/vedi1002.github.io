IoT Gas Sensing

IoT Gas Sensing was a group project that focused on building a gas monitoring and alarm system with the mbed, Raspberry Pi, and MQ gas sensors. The user can monitor near real-time gas data from multiple devices with different MQ sensors on the website GUI. Additionally, the user can set thresholds for specific sensors on the website GUI and the system would alert them via SMS and email if the thresholds had been breached. The notification system would specify which sensors had been triggered.
